## TOOLS(非前端)

### 添加自动触发工具
- 该工具可被llm调用，能让其在指定时间或条件下触发该session的运行（触发器触发后，向指定 session 注入一条 trigger_message，并启动一次受控的 session run）
- 用户能够在右侧账户观察区域查看、修改和删除触发器
- 每个触发器只触发一次，触发后在前端右侧的列表中变灰（失效），但如果用户手动修改并应用，则自动重新激活
需要实现以下几个工具：
- 下次开盘/集合竞价触发器(可选对应阶段（如午市开盘或早市集合竞价），自动设定时间)
- 定时提醒(根据账户的模拟时间设定，可输入绝对时间或相对时间，后端计算出最终触发的模拟时间后创建对应触发器)
- 股票涨跌幅、价格等条件触发（需要支持表达式，例如连续涨跌）
- 列出触发器列表（不包含已触发的失效触发器）
- 删除触发器

trigger_message需要包含以下字段：
- 触发器类型
- 触发条件/原因
- note（在创建触发时要求填写）
- 当前时间（模拟时间）
